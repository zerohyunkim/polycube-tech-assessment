## 실행 방법

1. 이 저장소를 클론하고 디렉토리로 이동합니다.

```sh
git clone https://github.com/zerohyunkim/polycube-tech-assessment.git
cd polycube-tech-assessment
```

2. 의존성을 설치하고 실행합니다.

```sh
pnpm i # 또는 npm i
pnpm dev
```
